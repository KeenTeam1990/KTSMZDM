## 简要说明

使用Swift4.1开发

使用SnapKit布局页面

完全自定义的NavigationBar和Tabbar适配全屏幕

自定义的MJRefresh下拉刷新动画控件

使用RxSwift绑定视图上的事件和网络请求

使用Alamofire+Moya/RxSwift+Codable轻松从Json转到Model

集成友盟和微博、微信、QQ分享等功能
